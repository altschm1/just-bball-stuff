Glossary

Basic

Stats from this page provide tracking stats on all matchup data through the course of the season.

Stats from Basic tab are all of the same stats found at https://www.nba.com/stats/player/{unique_nba.com_player_id}/head-to-head/ with a few additional metrics.

For more information on how nba.com tracks and record these matchup assignments, check out this link here.

Some notes:

  • Matchup data here does account for switches that take place during a defensive possession when calculating the number of partial possessions.
  • Shot, assists, and turnovers are assigned to whoever was defending the offensive player at that moment.
  • Double teams and help defense is accounted for, so if a shot or assist or turnover is taken when guarded by two defenders, both defenders will be credited with that shot, assist, or turnover.
  • Matchup stats only account for what happens in the front-court, not the backcourt.
  • These changes were implemented in 2019-20, whether they went back and made these changes for 2017-18 and and 2018-19, I do not know.

SEASON Regular Season
MINS the Number of Minutes and Seconds that OFF PLAYER and DEF PLAYER were matched up against each other as calculated by nba.com (see the above link for methodology)
POSS the Number of Partial Possessions that OFF PLAYER and DEF PLAYER were matched up against each other as calculated by nba.com (see the above link for methodology)
PTS the Number of Points that OFF PLAYER scored when matched up against DEF PLAYER
SHOT ATTEMPTS* the Number of Shots* that OFF PLAYER took when matched up against DEF PLAYER
This is an estimated number of shot attempts that is calculated by the following:
2PA + 3PA + (0.44 * FTA)
The 0.44 * FTA is how FTAs are typically converted to shot attempts when factoring in and-ones and fouled three-pointers, hence shot attempts* is only an estimate of the actual number of shots taken.
2PM the Number of Two Pointers Made by OFF PLAYER when matched up against DEF PLAYER
2PA the Number of Two Pointers Attempted by OFF PLAYER when matched up against DEF PLAYER
2P% the Number of Two Pointers Made divided by the number of Two Point Attempts
2PM / 2PA
3PM the Number of Three Pointers Made by OFF PLAYER when matched up against DEF PLAYER
3PA the Number of Three Pointers Attempted by OFF PLAYER when matched up against DEF PLAYER
3P% the Number of Three Pointers Made divided by the number of Three Point Attempts
3PM / 3PA
FTM the Number of Free Throws Made by OFF PLAYER when fouled/defended by DEF PLAYER
FTA the Number of Free Throws Attempted by OFF PLAYER when fouled/defended by DEF PLAYER
AST the Number of Assists by OFF PLAYER when matched up against DEF PLAYER
TOV the Number of Turnovers by OFF PLAYER when matched up against DEF PLAYER
eFG% the Effective Field Goal Percentage by OFF PLAYER when matched up against DEF PLAYER
(2PM + 1.5 * 3PM) / (2PA + 3PA)
TS% the True Shooting Percentage by OFF PLAYER when matched up against DEF PLAYER
(FTM + 2 * 2PM + 3 * 3PM) / (2 * (2PA + 3PA + (0.44 * FTA)))
3PAr the Three Point Attempt Rate by OFF PLAYER when matched up against DEF PLAYER
(3PA) / (2PA + 3PA)
FTr the Free Throw Rate by OFF PLAYER when matched up against DEF PLAYER
(FTA) / (2PA + 3PA)

Stats

Stats on the tab Stats are aggregate statistics that were created by me to help better contextualize the role of the defensive players.

These stats are mainly in a first iteration for me and will need to be improved upon.

A lot of the idea behind these stats were inspired by the excellent work done by Tim and Krishna Narsu at Bball-Index (https://www.bball-index.com/) where they have already put in a lot of work towards measuring and quanitifying Defensive Roles, Matchup Difficulty, and Matchup Versatility.

My work here isn't meant as an alternative or improvement to their work (being that this is only my first iteration, it is most definitely not an improvement at this stage). Rather, this is my attempt to get my feet wet with working with this type of data.

I will eventually write posts that explain the rationale and methodology behind each of these metrics, flaws and biases of these metrics, what can and needs to be improved, and what can and can't be taken away from these metrics.

But for now, I will give a brief overview of each of these metrics.

USG SCORE Usage Score
a proxy for the matchup difficulty faced by a player
measured on a scale of 0 to 100
Note: This does not necessarily measure how good a defender is at defending those high usage players!
Methodology
Give each offensive player an adjusted offensive usage score: (2PA + 3PA + (0.44 * FTA) + TOV) * OFF POSS**-0.5
For each defensive player, find the weighted average of the adjusted offensive usage score each defensive player played against for the entire season where weights are the number of POSS played against each offensive player
For each season, convert adjusted scores to percentiles on a scale from 0 to 100.
VERSATILITY SCORE Versatility Score
a proxy for how many different sized player a defender is asked to defend
measured on a scale of 0 to 100
Note: This does not necessarily measure how good a defender is at switching!
Methodology
Measure the frequency of how often a defender measures 4 different type of offensive players
Players 6'3 and shorter; Players 6'4 to 6'6; Players 6'7 to 6'9; Players 6'10 and taller
Based on these frequency, calculate the Shannon entropy: H(X) = −∑P(xi)logP(xi)
Players with a more even distribution between the four groups will have a higher Shannon entropy.
For each season, convert these vertsatility scores to percentiles on a scale from 0 to 100.
